# 国家試験対策ツール

国家試験の過去問PDFを活用した学習支援Webアプリケーション。

## 機能

- **PDF自動解析**: 過去問PDFから問題・選択肢・解説を自動抽出
- **別冊画像自動紐付け**: 問題文から別冊参照を検出し、対応画像と自動紐付け
- **学習モード**: ヒント機能付きで理解を深める学習
- **テストモード**: 制限時間付きで本番を想定した実力測定
- **結果レポート**: PDF形式で結果を出力（NotebookLM対応）

## 技術スタック

- React + TypeScript
- Vite
- IndexedDB (idb)
- PDF.js (pdfjs-dist)
- Tailwind CSS
- Zustand

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発モードで起動（ブラウザで自動的に開きます）
npm run dev

# ビルド
npm run build

# ビルド結果をプレビュー
npm run preview
```

## 使用方法

1. ブラウザでアプリを開く（`npm run dev`実行後、自動的に開きます）
2. 「PDFファイルを選択」ボタンで過去問PDFファイルを選択（複数選択可）
3. モードを選択（学習モード or テストモード）
4. 出題設定（回次、時間帯、出題数）
5. 「学習を開始する」または「テストを開始する」ボタンをクリック

## デプロイ

```bash
# ビルド
npm run build

# distフォルダをWebサーバーにデプロイ
# 例: Netlify, Vercel, GitHub Pagesなど
```

## ファイル命名規則

過去問PDFファイルは以下の命名規則に従う必要があります：

- 問題PDF: `{年度}_{回次}_{時間帯}.pdf`
  - 例: `2021_29_gozen.pdf` (2021年度 第29回 午前)
- 別冊PDF: `{年度}_{回次}_{時間帯}_bessatsu.pdf`
  - 例: `2021_29_gozen_bessatsu.pdf` (2021年度 第29回 午前 別冊)

時間帯:
- `gozen` = 午前
- `gogo` = 午後
